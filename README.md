# ambiorix-cli

<img src="https://ambiorix.john-coene.com/_assets/ambiorix.png" height = "200px"/>

Command line interface for [ambiorix](https://ambiorix.john-coene.com).

</div>

## Install

Install globally.

```bash
npm install ambiorix-cli -g
```

## Usage

Creates application skeleton.

```bash
ambiorix-cli myapp
```
